Installation Instructions

  1. Copy the embed code above.
  2. Open your website's HTML template or CMS theme editor.
  3. Paste the code immediately before the </body> tag.
  4. Save and publish your changes.
  5. Visit your site and look for the ACC accessibility icon in the bottom-right corner.
  6. Return to this dashboard — your widget status will update within 15 minutes.

✉ Zoho Email Automation

3 steps to activate automated compliance reminders from accessibility@[yourdomain].ca.

1
Verify Sending Domain in Zoho Mail
  1. Go to mail.zoho.com → Settings → Mail Accounts
  2. Add accessibility@[yourdomain].ca as sending alias
  3. Add SPF record in Canspace DNS: v=spf1 include:zoho.com ~all
  4. Add the DKIM record Zoho provides to Canspace DNS
2
Set Up Zoho Flow Scheduled Triggers
  1. Go to flow.zoho.com → Create New Flow
  2. Trigger: Schedule → December 1 annually at 9:00 AM EST
  3. Action: Zoho Mail → Send Email from accessibility@[yourdomain].ca
  4. Duplicate flow for Dec 15 (2nd reminder) and Dec 28 (final)
3
Paste This Template into Zoho Flow
