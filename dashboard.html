🎗 Widget Features

Accessibility ProfilesShows 6 disability-optimized presets (Visually Impaired, Motor, Senior, ADHD, Deaf, Cognitive). Visitors click one to instantly apply a full set of adjustments.
Auto-Fix EngineWidget automatically patches detectable issues at runtime — missing alt text, skip links, empty ARIA labels. Does not require code changes on your site.
Colour & Contrast ToolsEnables colour filter swatches (high-contrast, greyscale, inverted) and contrast adjustment slider in the widget settings tab.
Reading ToolsEnables the Reading Guide (highlights the current line), Reading Mask (dims surrounding content), and text spacing controls.
Language SelectorShows the 🌐 globe button in the widget header. Visitors can translate the entire page into 90+ languages via Google Translate. EN/FR default is set in Appearance above.
Visitor Feedback TabAdds a Feedback tab in the widget where visitors can report accessibility issues directly. Submissions appear in your Feedback Logs panel.

📊 Audit Reports

Monthly WCAG 2.1 AA compliance reports generated automatically on the 1st of each month.

ⓘ How reports work
Your ACC widget scans each page as visitors browse your site and sends results to Supabase automatically. On the 1st of each month, all scans from that month are aggregated into a PDF — your compliance score covers your entire site across all pages scanned, not a single visit. Your Baseline Assessment is generated at onboarding and captured your site's Day 0 state. The Overview tab shows this month's aggregate score as a live indicator.

✉ Zoho Email Automation

3 steps to activate automated compliance reminders from accessibility@[yourdomain].ca.

1
Verify Sending Domain in Zoho Mail
  1. Go to mail.zoho.com → Settings → Mail Accounts
  2. Add accessibility@[yourdomain].ca as sending alias
  3. Add SPF record in Canspace DNS: v=spf1 include:zoho.com ~all
  4. Add the DKIM record Zoho provides to Canspace DNS
2
Set Up Zoho Flow Scheduled Triggers
  1. Go to flow.zoho.com → Create New Flow
  2. Trigger: Schedule → December 1 annually at 9:00 AM EST
  3. Action: Zoho Mail → Send Email from accessibility@[yourdomain].ca
  4. Duplicate flow for Dec 15 (2nd reminder) and Dec 28 (final)
3
Paste This Template into Zoho Flow
